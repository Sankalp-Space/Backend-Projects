Routes


User Authentication
    /signup: POST - Create a new user account
    /login: POST - Login in to an existing user account


Voting : 
    /candidate: GET - Get the list of elector.
    /vote/electorId : POST - Vote for a specific candidate.

Vote Counts:
    /vote/counts:GET - Get the list of the elector sorted by their vote counts.


User Profile:
    /profile: GET -Get the user profile information.
    /profile/password :PUT- Change the user's password.

Admin Candidate Management:
    /electors:POST - Create a new candidate.
    /electors/electorId :PUT - Update the existing candidate.
    /electors/electorId : DELETE- Delete the existing candidate from the list.
